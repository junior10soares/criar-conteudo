# Avatar 3D

Coloque aqui o arquivo `avatar.vrm` usado no overlay 3D com lip-sync dos vídeos.

- **Como gerar o seu**: crie/customize seu avatar no [VRoid Studio](https://vroid.com/en/studio)
  (gratuito) olhando uma foto sua como referência, exporte como `.vrm` e salve como
  `assets/avatar/avatar.vrm` (mesmo nome, substitui o placeholder).
- **Placeholder atual**: `Seed-san.vrm`, modelo de demonstração oficial do consórcio VRM
  (VirtualCast Inc., licença VRM Public License 1.0), usado só pra testar o pipeline.
- **Sem esse arquivo**: a etapa de avatar 3D é pulada silenciosamente e o vídeo sai normal, sem
  overlay — nunca é obrigatório.
